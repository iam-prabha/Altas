# Resources

High-quality resources for learning RAG. Sourced before each lesson.

## Papers

- [Lewis et al. 2020 — "Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks"](https://arxiv.org/abs/2005.11401) — The original RAG paper. Defines the paradigm.
- [Gao et al. 2023 — "Retrieval-Augmented Generation for Large Language Models: A Survey"](https://arxiv.org/abs/2312.10997) — Comprehensive survey covering Naive → Advanced → Modular RAG paradigms, evaluation, and future directions.

## Guides & Tutorials

- [Pinecone — "RAG" learning series](https://www.pinecone.io/learn/rag/) — Clear visual explanations of each component.
- [LangChain RAG tutorial](https://python.langchain.com/docs/tutorials/rag/) — Framework-based walkthrough (use after fundamentals).
- [LlamaIndex RAG starter](https://docs.llamaindex.ai/en/stable/getting_started.html) — Alternative framework.

## Tools we use

- [sentence-transformers](https://www.sbert.net/) — Embedding models for Python.
- [FAISS](https://github.com/facebookresearch/faiss) — Vector similarity search library.
- [Ollama](https://ollama.ai/) — Run LLMs locally.
- [OpenAI API](https://platform.openai.com/) — Cloud LLM provider.

## Communities

- [r/RAG](https://www.reddit.com/r/RAG/) — Reddit community focused on RAG systems.
- [r/LocalLLaMA](https://www.reddit.com/r/LocalLLaMA/) — Local model community, good for Ollama users.
