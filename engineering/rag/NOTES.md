# Notes & Preferences

## Learning preferences
- Sessions: 45–60 min
- Wants both conceptual understanding AND working code
- Flexible on LLM: design for Ollama + OpenAI, let user choose

## Tooling
- Python primary
- No vector DB preference — start simple (FAISS), graduate later
- Don't use heavy frameworks (LangChain, LlamaIndex) in early lessons — build from scratch for understanding
