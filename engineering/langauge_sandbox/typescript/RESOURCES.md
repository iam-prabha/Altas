# TypeScript Resources

## Knowledge

- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html)
  The official documentation. Primary source for all language features. Always start here.
- [TypeScript Deep Dive — Basarat](https://basarat.gitbook.io/typescript/)
  Free, community-driven book. Good for practical patterns and "why" explanations behind the type system.
- [Matt Pocock — Total TypeScript](https://www.totaltypescript.com/)
  Modern, practical tutorials focused on real-world TypeScript patterns. Strong on generics and advanced types.
- [Execute Program — TypeScript Basics](https://www.executeprogram.com/courses/typescript-basics)
  Spaced-repetition interactive course. Good for building long-term retention of syntax and concepts.
- [TypeScript Playground](https://www.typescriptlang.org/play/)
  Official interactive playground. Useful for experimenting with types without setting up a project.

## Wisdom (Communities)

- [TypeScript Discord](https://discord.gg/typescript)
  Official community. Good for troubleshooting and pattern discussions.
- [r/typescript](https://reddit.com/r/typescript)
  Active subreddit for news, questions, and pattern discussions.
- [TypeScript GitHub Discussions](https://github.com/Microsoft/TypeScript/discussions)
  High-signal discussions on language design decisions and best practices.
